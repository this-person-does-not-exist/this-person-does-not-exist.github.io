# This Person Does Not Exist Browser

A static Tailwind CSS and Alpine.js gallery that loads 20 images per page from:

```text
https://thispersondoesnotexist.com
```

Open `index.html` in a browser, or serve the folder with any static web server.

The Previous, Next, and Reload controls perform full-page navigation or reloads so the browser requests a fresh page of image elements.
